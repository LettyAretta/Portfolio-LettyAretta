ESP WROOM 32 Project: Light Control

This project uses an LDR (Light Dependent Resistor) with the ESP32 to control an LED based on ambient light levels. The ESP32 reads the LDR's light intensity and turns the LED on or off depending on whether the light level exceeds a set threshold. This simple automation showcases the use of the ESP32 for real-time light monitoring and LED control.
